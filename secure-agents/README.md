# Securing AI Agents with IBM watsonx Orchestrate, IBM Verify, and HashiCorp Vault

This repository accompanies a technical article and hands-on tutorial on mitigating the **Confused Deputy Problem** in agentic AI systems. It demonstrates how to secure tool-calling AI agents using IBM watsonx Orchestrate, IBM Verify (OAuth 2.0 token propagation), and HashiCorp Vault (secrets management) — preventing agents from acting beyond the permissions of the user who invoked them.
