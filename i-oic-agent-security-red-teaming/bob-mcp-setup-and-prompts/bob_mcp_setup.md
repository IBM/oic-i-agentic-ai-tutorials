# Setting Up IBM BOB with the watsonx Orchestrate ADK Docs MCP Server

This allows BOB to leverage the watsonx Orchestrate ADK Docs MCP server to answer watsonx Orchestrate–related queries and access relevant documentation and knowledge.


## Step 1
1. Open **IBM BOB**.  
2. On the right-hand panel, in the BOB chat, click the **three-dot menu (⋯)** located at the top-right corner.

![image](https://github.ibm.com/user-attachments/assets/cb583646-96a3-4246-8d03-acb3e2c6608f)

---

## Step 2
Click **MCP Servers**.

![image](https://github.ibm.com/user-attachments/assets/236ba4aa-9f45-4499-8a3b-c4f9f6653590)

---

## Step 3
In the search panel, search for **"watsonx Orchestrate ADK Docs"**.  
Click **Install** as shown in the screenshot.

![image](https://github.ibm.com/user-attachments/assets/891e53fd-a4e5-454e-aae5-297fcfc49c55)


---

## Step 4
Select **Global**, then click **Install** as shown in the screenshot.

![image](https://github.ibm.com/user-attachments/assets/c2448b34-028c-4ef7-b97d-d7e03083df05)

---

## Step 5
Once installed, go to the BOB chat and ask:

```
List the MCP servers that you have access to.
```

Verify that **watsonx-orchestrate-adk-docs** appears in the response.  

![image](https://github.ibm.com/user-attachments/assets/9f6aa432-5b3f-403b-8142-9015fa759b68)
![image](https://github.ibm.com/user-attachments/assets/f6358823-e940-4864-87cd-b7be135c4f69)


This confirms the MCP server is successfully installed and ready to use for watsonx Orchestrate-related queries.
